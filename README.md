# LaboTrack

## Description du projet
LaboTrack est une application de gestion des échantillons médicaux destinée aux laboratoires.
Elle permet de suivre les échantillons, leur stockage (frigos, box), les catégories et les utilisateurs.
Ce projet est réalisé dans le cadre d’un projet de fin de stage en Génie Logiciel.

## Objectifs du projet
- Digitaliser la gestion des échantillons médicaux
- Réduire les erreurs de suivi manuel
- Améliorer la traçabilité et la sécurité des données
- Gérer les utilisateurs selon leurs rôles (admin, personnel, visiteur)

##  Fonctionnalités principales
- Authentification des utilisateurs
- Gestion des utilisateurs (admin)
- Gestion des frigos, box et catégories
- Gestion des échantillons
- Consultation des échantillons pour les visiteurs
